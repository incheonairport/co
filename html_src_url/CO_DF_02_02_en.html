TOBT (목표 주기장 출발시간)

주기장에서 aircraft의 출발준비가 Completed되어 계류장관제소의 승인을 받으면 즉시 PUSHBACK을 할 수 있다고 Airline/조업사가 예상하는 목표시간

All Doors Closed
Boarding Bridges 이현
토잉차량 준비
조종사 출발준비 완료
TOBT(목표 주기장 출발 시간)는 Airline/조업사에서 입력

목표신뢰도: +/-5분 이내 (EOBT와 15분 이상 차이 발생 시 EOBT Revision)

  • Control tower TSAT 발부 전, 제한없이 입력/Revision
  • Control tower TSAT 발부 후, 3회 Revision 제한 또는 EOBT 변경
  • 계류장 관리소에 유선으로 요청
TSAT(목표 PUSHBACK 승인시간)
  • Control tower에서 TOBT를 기초로 산출된 TTOT에 Above-ground이동시간을 계산하여 제공하는 엔진시동 및 PUSHBACK 승인발부 예상시간
  • 시스템자동산출 TSAT TOBT를 기반으로 자동 산출된 시간표출(참조용)
  • Control tower 제공 TSAT ACC와 TWR간 TTOT 협의 후 산출된 시간으로서 파란색으로 구분 표출
  • TSAT issue
  • TOBT 지연
  • TSAT 재발부
  • EXOT
TTOT(목표Take off시간)
  • Control tower에서 TOBT 및 CTOT, ATFM, 항로, 고도, 교통흐름을 고려하여 Take off순사를 예측함으로써 제공하는 목표Take off시간
  • 시스템자동산출 TTOT TOBT를 기반으로 자동 산출된 시간표출(참조용)
  • Control tower 제공 TTOT ACC와 TWR간 협의된 이륙시간으로서 파란색으로 구분 표출
  • TSAT issue
