TOBT (目标停机场出发时间)

航空公司/飞机的目标时间是在飞机上完成了飞机的出发准备，如果得到停机坪的批准，就可以立即进行pushback

토잉차량 준비
조종사 출발준비 완료
TOBT(目标停机场出发时间)是航空公司/助业公司输入

目标信赖度: +/-5分以内 (EOBT和 发生15分以上差异时修改EOBT )

  • 控制塔 TSAT 发布之前，无限制的输入/修正
  • 控制塔 TSAT 发布后, 3次修改限制或变更 EOBT
  • 向停机坪管理所发出有线要求
TSAT(目标 PUSHBACK 批准时间)
  • …在控制塔上以tobt为基础计算的ttot上计算出地上移动时间的发动机启动及pushback认证的预测时间
  • 系统自动计算TSAT 以 TOBT为基准自动计算的时间表示(参照用)
  • 提供控制塔 TSAT ACC与TWR间 TTOT 协议后计算出来的时间 用 蓝色区分表示
  • TSAT issue
  • TOBT 지연
  • TSAT 재발부
  • EXOT
TTOT(目标起飞时间)
  • 在控制塔上，考虑到tobt、ctot、atfm、航线、高度、交通流量，以预测起飞的巡查时间的目标起飞时间
  • 系统自动计算TSAT 以 TOBT为基准自动计算的时间表示(参照用)
  • 提供控制塔 TSAT ACC与TWR间 TTOT 协议后计算出来的时间 用 蓝色区分表示
  • TSAT issue
